# deleteOldFiles
Delete old files based on a specific month, week, and day from FTP 

## config
for run deleteOldFiles put your config in path `configs/configs.json`
```json
{
  "version": 1,
  "ftp_account": {
    "address": "127.0.0.1:21",
    "username": "",
    "password": "",
    "deletable_files_path":""
  }
}
```
